# Projeto Prático - Curso de Formação em PHP

Este repositório contém um projeto prático desenvolvido durante o **curso de Formação em PHP** da Rocketseat.

O objetivo do projeto é aplicar conceitos fundamentais de desenvolvimento backend utilizando **PHP**, com foco em **arquitetura MVC** e **integração com banco de dados**.

## 📚 Conceitos Aplicados

- Estrutura de projeto baseada em **MVC (Model - View - Controller)**
- Organização de código seguindo boas práticas
- Conexão e manipulação de **banco de dados**
- Separação de responsabilidades entre camadas
- Estruturação de rotas e controle de requisições

## 🛠 Tecnologias Utilizadas

- PHP
- MySQL
- HTML
- CSS

## 🎯 Objetivo do Projeto

Praticar e consolidar conceitos importantes do desenvolvimento backend, principalmente:

- Organização de projetos PHP
- Implementação da arquitetura **MVC**
- Manipulação de dados com banco de dados
- Estruturação de aplicações web escaláveis
